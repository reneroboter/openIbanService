
openIbanService
========

jQuery wrapper for easily calling openiban.org json services.


How to use
----------
```javascript
$('#myIbanField').getBankAndBicWithIBAN({
        targetBic: $('#myTargetBIC'),
        targetBank: $('#myTargetBank')
    });
```
